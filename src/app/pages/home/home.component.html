Sobre el Proyecto

Esta API proporciona datos sobre resultados deportivos en tiempo real, incluyendo información sobre partidos, equipos y estadísticas. Fue creada para facilitar la integración de datos deportivos en aplicaciones web y móviles.

Tecnologías Utilizadas

Frontend: Angular

Backend: Node.js, Express

Api: thesportsdb.com

Historia del Proyecto

Este proyecto comenzó como un reto personal para aprender más sobre integración de APIs en Angular. A medida que avanzaba, se incluyeron más funcionalidades, como resultados en vivo de deportes, y la base de datos fue mejorando con el tiempo.
